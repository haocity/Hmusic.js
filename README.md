# Hmusic.js
![log.png](https://ooo.0o0.ooo/2017/06/03/5932d9fcb06da.png)
简单的音乐播放器

## 介绍
![demo.png](https://ooo.0o0.ooo/2017/06/03/5932daa9d4bab.png)
- 可播直接放网易云歌单  支持歌词滚动
### 食用方法
引用hmusic.js
```
<script  src="hmusic.js" type="text/javascript" charset="utf-8"></script>
```
播放云歌单
```
hyplaylist(Element,YUNID);
```
Element:播放器所要放入的元素
YUNID：网易云歌单ID
